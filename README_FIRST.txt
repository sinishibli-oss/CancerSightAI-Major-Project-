CancerSight AI — Quick Start (Windows)

1. Install Python 3.10 or 3.11.
2. Extract this ZIP.
3. Open the extracted folder.
4. Click the File Explorer address bar, type cmd, and press Enter.
5. Run: python -m venv venv
6. Run: venv\Scripts\activate
7. Run: python -m pip install --upgrade pip
8. Run: pip install -r requirements.txt
9. Run: python app.py
10. Open http://127.0.0.1:5000 in Chrome.

Upload a JPEG, PNG, TIFF, or WEBP histopathology image (maximum 10 MB).
This application is for academic demonstration only and is not a medical diagnostic tool.
