AI-Based Multi-Cancer Histopathology Classification

CancerSight AI is an explainable deep learning platform developed for automated histopathology image classification. Upload a lung, colon, or breast tissue image to receive the predicted cancer class, confidence score, and a Grad-CAM visual explanation showing the tissue regions that influenced the model's decision.

About CancerSight AI

CancerSight AI is an explainable deep learning framework developed as part of a Master of Computer Applications (Artificial Intelligence) major project. The system classifies histopathology images using deep learning while providing Grad-CAM visual explanations to improve transparency.

Supported Cancer Types

Seven classification outcomes across lung, colon, and breast histopathology.

01

Lung Cancer

  • Adenocarcinoma
  • Squamous Cell Carcinoma
  • Normal
02

Colon Cancer

  • Normal
03

Breast Cancer

  • IDC Positive
  • IDC Negative
